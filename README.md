# Pizza Sales Analysis

## Project Overview

This project analyzes pizza store sales data to identify key business insights such as revenue trends, customer purchasing behavior, and top-performing products. The goal is to support better decision-making in sales, marketing, and operations.

## Objectives

* Calculate total revenue, total orders, and pizzas sold
* Identify best-selling and least-selling pizzas
* Analyze sales trends by day, month, and hour
* Understand performance by pizza category and size
* Compute key KPIs like Average Order Value (AOV)

## Dataset

**File:** `pizza_sales.csv`

**Key Columns:**

* `order_id` – Unique order identifier
* `pizza_name` – Name of the pizza
* `quantity` – Number of pizzas sold
* `total_price` – Revenue per order
* `date`, `time` – Order timestamp
* `pizza_category`, `pizza_size` – Pizza classification

## Key KPIs

* **Total Revenue**
* **Total Orders**
* **Total Pizzas Sold**
* **Average Order Value (AOV)**
* **Average Pizzas per Order**

##  Analysis Performed

* Daily, monthly, and hourly sales trends
* Category-wise and size-wise sales analysis
* Top 5 best-selling pizzas
* Bottom 5 least-selling pizzas
* Data visualizations using charts and graphs

##  Tools & Technologies

* Python
* Pandas
* Matplotlib / Seaborn
* Jupyter Notebook

##  Key Insights

* Certain pizza categories and sizes contribute most to revenue
* Sales peak during specific hours and days
* A few pizzas dominate overall sales while some underperform

---

## Conclusion

This analysis helps understand sales performance and customer preferences, enabling better inventory planning, promotions, and business strategy.
